# Sight

A cross-platform image viewer.

Features:
- Select image: Ctrl + Click
- Get image info: Hovering mouse over image

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Pack package
```
npm run package
```

### Make package
```
npm run make
```

### Lints files
```
npm run lint
```
